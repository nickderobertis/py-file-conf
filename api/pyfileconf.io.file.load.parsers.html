Identifies the root part of an assignment value, so long as it is not builtin.

Examples from a import b

b -> ‘b’ b.attr -> ‘b’

import c

c.test -> ‘c’ c.thing.stuff -> ‘c’

‘abc’ -> None 123 -> None
